前言

这一章节将给拓扑空间中任意一点定义它的邻域。点 xx 的邻域从直观上看是指与 xx 十分接近的点所组成的集合，但实际上这个集合并不一定很小。不过，在实际应用上更重要的是相对比较小的邻域，并且还提出了邻域基的定义。所谓邻域基，简单来说就是所有邻域中最具代表性的那些邻域所组成的集族。

正文

定义2.1（邻域）

设 XX 为拓扑空间， x∈Xx\in X 。如果集合 V⊆XV\subseteq X 满足：存在开集 U⊆XU\subseteq X ，使得 x∈U⊆Vx\in U\subseteq V ，那么 VV 叫作点 xx （在空间 XX 中）的邻域。如果 VV 恰好是开集（闭集），那么 VV 叫作 xx 的开（闭）邻域。

从定义可以看出， XX 本身也是 xx 的一个邻域（所以邻域不一定非常小），然而实际问题中讨论这种较大的邻域通常没有意义，小邻域更能体现问题的本质。在 ε−δ\varepsilon-\delta 语言中也是同理，取任意小的 ε\varepsilon 或 δ\delta 更加本质。

某一点的开邻域其实就是含有该点的开集（从而开集是它每一点的邻域），即以下命题成立。

命题2.2（开邻域的等价描述）

设 XX 为拓扑空间， x∈Xx\in X 。对于 V⊆XV\subseteq X ，以下两个命题等价：

  1. VV 是 xx 的开邻域。
  2. VV 是开集，并且 x∈Vx\in V 。

证明：先证明 1⇒21\Rightarrow2 。如果 VV 是 xx 的开邻域，那么 VV 就是 XX 的开集。且存在开集 U⊆XU\subseteq X ，使得 x∈U⊆Vx\in U\subseteq V ，即 x∈Vx\in V 也成立。

再证明 2⇒12\Rightarrow1 。设 VV 是开集，并且 x∈Vx\in V 。此时只要令 U=VU=V ，就有存在开集 U⊆XU\subseteq X ，使得 x∈U⊆Vx\in U\subseteq V 成立，所以 VV 是开邻域。

证毕。

邻域的有限个交集仍然是邻域。

命题2.3（邻域的有限个交集）

设 XX 为拓扑空间， x∈Xx\in X 。对于任意正整数 nn 和 xx 的邻域 V1,V2,⋯,Vn⊆XV_1,V_2,\cdots,V_n\subseteq X ，交集 ⋂i=1nVi\bigcap_{i=1}^n V_i 也是 xx 的邻域。

证明：根据邻域的定义，对于每个 i=1,2,⋯,ni=1,2,\cdots,n ，存在开集 UiU_i 满足 x∈Ui⊆Vix\in U_i\subseteq V_i 。于是 ⋂i=1nUi\bigcap_{i=1}^nU_i 也是开集，且 x∈⋂i=1nUi⊆⋂i=1nVix\in\bigcap_{i=1}^nU_i\subseteq\bigcap_{i=1}^nV_i ，因此 ⋂i=1nVi\bigcap_{i=1}^n V_i 也是 xx 的邻域。

证毕。

下面的命题是证明拓扑空间的子集为开集的常用手段。

命题2.4（开集的判定）

设 X 为拓扑空间， U\subseteq X ，以下两个命题等价：

  1. U 是开集。
  2. \forall x\in U ，存在 x 的邻域 V ，使得 V\subseteq U 。

证明：先证明 1\Rightarrow2 。设 U 是开集，任取 x\in U ，根据命题2.2， U 是 x 的开邻域。此时只要令 V=U ，就有存在开邻域 V ，使得 V\subseteq U 。

再证明 2\Rightarrow1 。设条件2成立，对于每个 x\in U 都能找到相应的邻域 V(x) ，使得 V(x)\subseteq U 。又根据邻域的定义，能找到相应的开集 U(x)\subseteq V(x)\subseteq U 。此时， U=\bigcup_{x\in U}U(x) ，因此 U 是开集。

证毕。

点 x 的邻域通常情况下是各种各样的，处理起来很麻烦，所以我们经常只考察某些特定的邻域，这种对象就叫作 x 的邻域基。

定义2.5（邻域基）

设 X 为拓扑空间， x\in X 。如果由 x 的部分邻域组成的集族 \mathcal{U} 满足：对于 x 的任意邻域 V ，总是存在邻域 U\in \mathcal{U} ，使得 U\subseteq V ，就把 \mathcal{U} 叫作 x 的邻域基。

注意，定义中的任意邻域 V 也可以改成任意开邻域 V ，这一点请大家自己证明。

例2.6（所有邻域组成邻域基）

这个例子有点无聊，由定义，点 x 的所有邻域组成了 x 的一个邻域基。

例2.7（所有开邻域组成邻域基）

设 X 为拓扑空间， x\in X 。设所有开邻域组成的集族为 \mathcal{U} ，对于 x 的任意邻域 V ，根据邻域的定义，存在开集 U 满足 x\in U\subseteq V 。又根据命题2.2可知 U 是 x 的开邻域，所以 \mathcal{U} 就是 x 的邻域基。

例2.8（距离空间中的邻域基）

设 (X,d) 为距离空间，对于 x\in X ，以 x 为中心的所有开球组成了 x 的邻域基。首先，以 x 为中心的开球 B(x,r) 是包含了点 x 的开集（命题1.16）。其次，设 V 为 x 的任意开邻域，此时存在开球 B(x,r)\subseteq V （定义1.15）。根据定义，所有 B(x,r) 组成了 x 的邻域基。

另外，半径为 \frac{1}{n},n\in\mathbb N_+ 的所有开球组成的集族也是 x 的邻域基。首先，开球 B(x,\frac{1}{n}) 是 x 的开邻域。其次，设 V 为 x 的任意开邻域，此时存在开球 B(x,r)\subseteq V 。又根据实数的阿基米德性质，存在 n\in\mathbb N_+ 使得 \frac{1}{n}<r ，所以 B(x,\frac{1}{n})\subseteq B(x,r)\subseteq V ，从而所有 B(x,\frac{1}{n}) 组成的集族就是 x 的邻域基。

例2.9（离散空间中的邻域基）

设 X 为离散空间，对于任意一点 x\in X ，集合 \left\{ x \right\} 是它的开邻域，所以由 \left\{ x \right\} 组成的集族 \left\{ \left\{ x \right\} \right\} 就是它的邻域基。

从直观上看， x 的邻域中的点都是与 x “相邻”的点。而在离散空间中，任意单点集 \left\{ x \right\} 都是 x 的开邻域，即没有其他点与 x 相邻，这就是离散空间名字的由来。一般地，在拓扑空间 X 中，如果 \left\{ x \right\} 是点 x 的邻域，就把 x 叫作 X 的孤立点。

设拓扑空间 X 中每一点 x 都具有邻域基，此时成立以下性质。

命题2.10（邻域基的性质）

设 X 为拓扑空间， x\in X ，它的邻域基为 \mathcal{U}_x 。此时成立以下4条性质：

  • NB1 ：对于任意的 x\in X,\mathcal{U}_x\ne\emptyset 。
  • NB2 ：对于任意的 x\in X,U\in\mathcal{U}_x ，有 x\in U 。
  • NB3 ：对于任意的 x\in X,U_1,U_2\in\mathcal{U}_x ，存在 V\in\mathcal{U}_x ，使得 V\subseteq U_1\cap U_2 。
  • NB4 ：对于任意的 x\in X,U\in\mathcal{U}_x ，存在 V\in\mathcal{U}_x ，它满足： \forall y\in V,\exists W\in\mathcal{U}_y （指点 y 的邻域基），使得 W\subseteq U 。

证明： NB1,NB2 根据定义可以马上验证，仿照命题2.3可证明 NB3 。

最后证明 NB4 。设 x\in X,U\in\mathcal{U}_x ，因为 U 是 x 的邻域，所以存在开集 U_0 满足 x\in U_0\subseteq U 。又因为 U_0 是 x 的开邻域（命题2.2），所以存在 V\in\mathcal{U}_x ，使得 V\subseteq U_0 。现在证明这个 V 就是 NB4 中所要找的 V ，为此，任取 y\in V 。注意 y\in V\subseteq U_0 ，而 U_0 是开集，所以 U_0 也是 y 的开邻域。于是存在 W\in\mathcal{U}_y ，使得 W\subseteq U_0\subseteq U 。

证毕。

虽然命题2.10比较复杂，但正如命题1.5所说我们可以通过闭集来定义拓扑空间那样，下面的命题2.11则表示通过给集合的每一点指定邻域基的方式同样可以定义拓扑空间。

命题2.11（从邻域基出发定义拓扑空间）

设 X 为非空集合，给每一点 x\in X 都指定一个满足条件 NB1\sim NB4 的集族 \mathcal{U}_x\subseteq P(X) 。此时，在 X 上有且只有一个拓扑 \mathcal{O} ，使得 \mathcal{U}_x 恰好为拓扑空间 (X,\mathcal{O}) 中点 x 的邻域基。

证明：定义集族：

\mathcal{O}=\left\{ U\subseteq X|\forall x\in U,\exists V\in\mathcal{U}_x,s.t.V\subseteq U \right\}

，我们来证明这个集族是 X 上的拓扑，即证明它满足条件 O1\sim O3 。

O1 ：先证 \emptyset\in\mathcal{O} ，即证 \forall x,x\in\emptyset\rightarrow \exists V\in\mathcal{U}_x\wedge V\subseteq\emptyset 。因为 x\in\emptyset 为假，蕴涵式 x\in\emptyset\rightarrow \cdots 为真，所以 \emptyset\in\mathcal{O} 。再证 X\in\mathcal{O} 。任取 x\in X ，条件 NB1 保证了至少存在一个 V\in\mathcal{U}_x ，而 V\subseteq X 是显然的，所以 X\in\mathcal{O} 。

O2 ：设 U_1,U_2\in\mathcal{O} ，交集为空时条件 O2 成立。交集不为空时，任取 x\in U_1\cap U_2 。对于每个 i=1,2 ，因为 U_i\in\mathcal{O} ，所以 \exists V_i\in\mathcal{U}_x ，使得 V_i\subseteq U_i 。根据条件 NB3 ，此时 \exists V\in\mathcal{U}_x ，使得 V\subseteq V_1\cap V_2\subseteq U_1\cap U_2 ，所以 U_1\cap U_2\in\mathcal{O} 。

O3 ：设 \Lambda 为指标集，集族 \left\{ U_\lambda|\lambda\in\Lambda \right\}\subseteq \mathcal{O} 。任取 x\in \bigcup_{\lambda\in\Lambda} U_\lambda ，则 \exists \lambda_0\in\Lambda 使得 x\in U_{\lambda_0} 。又因为 U_{\lambda_0}\in \mathcal{O} ，所以 \exists V\in \mathcal{U}_x 使得 V\subseteq U_{\lambda_0}\subseteq \bigcup_{\lambda\in\Lambda} U_\lambda ，于是 \bigcup_{\lambda\in\Lambda} U_\lambda\in\mathcal{O} 。

由此得到了拓扑空间 (X,\mathcal{O}) ，再证明 \mathcal{U}_x 为 x 的邻域基，由例2.6可知只需要证明 \mathcal{U}_x 中所有集合均为 x 的邻域。

任取 U\in\mathcal{U}_x ，定义集合 U'=\left\{ y\in U|\exists V\in\mathcal{U}_y ,s.t. V\subseteq U\right\} 。若取 V=U\in\mathcal{U}_x ，就可知 x\in U'\subseteq U 。为了说明 U'\in \mathcal{O} ，设 y\in U' 是任意一点，由 U' 的定义可知存在 V\in \mathcal{U}_y ，使得 V\subseteq U 。对 V 使用条件 NB4 ，存在 V'\in \mathcal{U}_y ，它满足：对于任意的 z\in V' ，总是存在 W\in \mathcal{U}_z ，使得 W\subseteq V\subseteq U 。而根据条件 NB2 ，有 z\in W\subseteq V ，由 z 的任意性得 V'\subseteq V\subseteq U 。进一步地，我们有 V'\subseteq U' ，这是因为已证得 z\in U 且存在 W\in \mathcal{U}_z 使得 W\subseteq U ，于是 z 满足 U' 的定义，由 z 的任意性得 V'\subseteq U' 。又由 y\in U' 的任意性得 U'\in \mathcal{O} 成立，这样就证明了 \mathcal{U}_x 中所有集合均为 x 的邻域，故 \mathcal{U}_x 为邻域基。

最后证明满足条件的 \mathcal{O} 是唯一的。假设 X 上有两个拓扑 \mathcal{O}_1,\mathcal{O}_2 都使得 \mathcal{U}_x 恰好为 x 的邻域基，任取 U\in\mathcal{O}_1 。对于任意的 x\in U ，因为 \mathcal{U}_x 为 x 的邻域基，所以存在 V\in\mathcal{U}_x ，使得 V\subseteq U 。另一方面， V\in\mathcal{U}_x 且 \mathcal{U}_x 是点 x 在拓扑空间 (X,\mathcal{O}_2) 的邻域基，所以 V 是 x 在拓扑空间 (X,\mathcal{O}_2) 的邻域。根据命题2.4， U 是拓扑空间 (X,\mathcal{O}_2) 的开集，即 U\in\mathcal{O}_2 ，所以 \mathcal{O}_1\subseteq\mathcal{O}_2 。同理可证 \mathcal{O}_2\subseteq\mathcal{O}_1 ，故 \mathcal{O}_1=\mathcal{O}_2 ，即这样的拓扑是唯一的。

证毕。

如果拓扑空间中每一点的邻域基都是至多可数集，这样的拓扑空间有专门的名字。

定义2.12（第一可数）

拓扑空间 X 是第一可数的，或者说满足第一可数公理，是指空间中的每一点都拥有至多可数的邻域基。

例2.13（距离空间是第一可数的）

距离空间是第一可数的。设 X 为距离空间，根据例2.8可知 \forall x\in X ，开球 B(x,\frac{1}{n}),n\in\mathbb N_+ 是 x 的一个可数邻域基。另外，根据例2.9可知离散空间也是第一可数的，因为单点集 \left\{ x \right\} 组成的集族 \left\{\left\{ x \right\}\right\} 作为 x 的邻域基也是至多可数的，当然这个结论也可以从离散空间可距离化（例1.20）得到。对于一般的可距离化空间，因为距离所诱导的拓扑为原来的拓扑，所以完全仿照上面的讨论就得到可距离化空间也是第一可数的。

不过，后面的例4.14中将给出一个虽然是第一可数空间，但不是可距离化空间的例子。

例2.14（有限补拓扑和第一可数性）

设 X 为无限点集，上面赋予了有限补拓扑（例1.8）。如果 X 为可数集，那么 X 是第一可数的。此时的 X 可以表示成点列 \left\{ x_n \right\} 的形式（当 i\ne j 时， x_i\ne x_j ），对于每一点 x_i ，易证任何补集为有限集的集合 U 并上 \left\{ x_i \right\} 都是它的邻域基，显然它是可数的。

而如果 X 为不可数集，那么 X 不是第一可数的。反证法，设 \exists x\in X ，它具有可数邻域基。给邻域基中每个邻域编号为 U_1,U_2,\cdots,U_n,\cdots ，则对于每个 U_i,i\in\mathbb N_+ ，存在开集 V_i 满足 x\in V_i\subseteq U_i 。根据有限补拓扑的定义，此时补集 V_i^c 为有限集，所以 U_i^c\subseteq V_i^c 也为有限集，从而并集 \bigcup_{i=1}^{\infty} U_i^c=(\bigcap_{i=1}^\infty U_i)^c 为至多可数集。而因为 X 不可数，所以交集 \bigcap_{i=1}^\infty U_i 也一定不可数。任取 y\in \bigcap_{i=1}^\infty U_i 且 y\ne x ，则集合 U=X-\left\{y \right\} 是 x 的开邻域。然而， y 属于每个 U_i ，但不属于 U ，所以不可能存在某个 U_i\subseteq U ，这就和邻域基定义矛盾。所以假设不成立， X 中不存在拥有可数邻域基的点，所以 X 不是第一可数的。另外， X 也一定不是可距离化空间，否则根据例2.13， X 可距离化就意味着它是第一可数的，矛盾。

为了以后的内容先提出以下命题。

命题2.15（第一可数空间邻域基的递减性）

设 X 为第一可数空间， x\in X 。此时存在 x 的可数邻域基 \left\{ V_n|n\in\mathbb N_+ \right\} ，使得 V_{n+1}\subseteq V_n 。

证明：根据第一可数的定义，点 x 存在可数邻域基，分别编号为 U_1,U_2,\cdots,U_n,\cdots 。定义集合 V_n=\bigcap_{i=1}^n U_i=U_1\cap U_2\cap\cdots\cap U_n ，根据命题2.3， V_n 是点 x 的邻域，且满足 V_{n+1}\subseteq V_n 。设 U 是 x 的任意邻域，则存在某个 U_n\subseteq U 。而由 V_n 的定义可知 V_n\subseteq U_n\subseteq U ，所以 \left\{ V_n|n\in\mathbb N_+ \right\} 是 x 的邻域基。

证毕。

接下来定义拓扑空间中点列收敛的概念，这是数轴 \mathbb R 上的数列收敛概念的推广。点列的收敛主要在第一可数空间（例如距离空间）中发挥威力，但是定义本身可以在一般的拓扑空间中进行。

拓扑空间 X 中的点列，用比较随意的说法则是 X 中的一列点 x_1,x_2,\cdots ，记作 (x_n )_{n=1}^\infty ，正式的定义则是由 f(n)=x_n 所定义的映射 f:\mathbb N_+\rightarrow X （点列的本质是映射）。有时我们也把点列记作 \left\{ x_n \right\} ，但这个符号同时也用来表示 f 的像集 \left\{ x\in X|\exists n\in \mathbb N_+,x=f(n) \right\} ，在使用中应当注意区分。

定义2.16（点列的收敛）

设 X 为拓扑空间， (x_n )_{n=1}^\infty 为 X 中的点列， x\in X 。如果对于 x 的任意邻域 V ，存在 N\in \mathbb N_+ ，当 n\geq N 时，有 x_n\in V ，就称点列 (x_n )_{n=1}^\infty 收敛于 x 。此时 x 叫作点列 (x_n )_{n=1}^\infty 的极限，记作 x_n\rightarrow x 。

另外，不难证明定义中的任意邻域 V 可换成任意开邻域 V ，意义不变（由例2.7和接下来的命题2.18也能看出）。

注意2.17（点列的极限不一定唯一）

一般来说，即使点列极限存在，它也未必是唯一的。例如平凡空间 X=\left\{0,1 \right\} ，定义点列 f(n)=0=x_n ，易证 x_n\rightarrow0,x_n\rightarrow1 。

后面（命题2.20）将会证明，距离空间中如果点列收敛，那么极限是唯一的。一般地，拓扑空间中的点列收敛至唯一极限时，写作 \lim_{n \rightarrow \infty}{x_n}=x 。

下面的命题告诉我们，要讨论点列在某一点是否收敛并不需要考虑任意邻域，而只需要考虑邻域基。

命题2.18（邻域基和点列的收敛）

设 X 为拓扑空间， (x_n )_{n=1}^\infty 为 X 中的点列， x\in X,\mathcal{U} 为 x 的邻域基。此时 (x_n )_{n=1}^\infty 收敛于 x 的充要条件是： \forall U\in\mathcal{U},\exists N\in\mathbb N_+ ，当 n\geq N 时，有 x_n\in U 。

证明：必要性是显然的，只证充分性。任取 x 的邻域 V ，因为 \mathcal{U} 是 x 的邻域基，所以存在 U\in\mathcal{U} 满足 U\subseteq V 。由假设， \exists N\in\mathbb N_+ ，当 n\geq N 时 x_n\in U\subseteq V 成立，所以 (x_n )_{n=1}^\infty 收敛于 x 。

证毕。

命题2.19（距离空间中点列的收敛）

设 X 为距离空间，对于 X 中的点列 (x_n )_{n=1}^\infty 和点 x\in X ，以下3个命题等价。

  1. x_n\rightarrow x 。
  2. \forall\varepsilon>0,\exists N\in\mathbb N_+ ，当 n\geq N 时， d(x_n,x)<\varepsilon 。
  3. d(x_n,x)\rightarrow0 ，即 \mathbb R 中的点列 (d(x_n,x))_{n=1}^\infty 收敛于0。

证明：先证 1\Rightarrow2 。因为 \left\{ B(x,r)|r>0 \right\} 是 x 的邻域基，根据命题2.18，条件1成立就意味着 \forall \varepsilon>0,\exists N\in\mathbb N_+ ，当 n\geq N 时， x_n\in B(x,\varepsilon) 。而 x_n\in B(x,\varepsilon) 也就意味着 d(x_n,x)<\varepsilon ，因此条件2成立。反过来即可证 2\Rightarrow1 ，所以1、2等价。

再证 3\Rightarrow2 。当 (d(x_n,x))_{n=1}^\infty\rightarrow0 时，利用1、2的等价性，此时 \forall \varepsilon>0,\exists N\in\mathbb N_+ ，当 n\geq N 时， |d(x_n,x)-0|=d(x_n,x)<\varepsilon ，这就是2。反过来即可证 2\Rightarrow3 ，所以2、3等价。

证毕。

命题2.20（距离空间中点列收敛则极限唯一）

设 X 为距离空间， (x_n )_{n=1}^\infty 为 X 中的点列， x,y\in X 。如果 x_n\rightarrow x,x_n\rightarrow y ，那么 x=y

证明：若 x\ne y ，记 r=d(x,y)>0 。因为 x_n\rightarrow x ，根据命题2.19， \exists N_1\in\mathbb N_+ ，当 n\geq N_1 时， d(x_n,x)<\frac{r}{2} 。同理， \exists N_2\in\mathbb N_+ ，当 n\geq N_2 时， d(x_n,y)<\frac{r}{2} 。取 N=\max\left\{ N_1,N_2\right\} ，则

r=d(x,y)\leq d(x,x_N)+d(y,x_N)<\frac{r}2+\frac{r}{2}=r

，矛盾。

证毕。

最后补充说明欧氏空间的子集中点列收敛的情况。设 A\subseteq\mathbb R^n ，则 A 在欧几里得距离的限制下（注意1.13）成为距离空间。下述命题告诉我们， A 中点列的收敛等价于按坐标收敛。

命题2.21（欧氏空间的子集中点列的收敛）

设 A\subseteq\mathbb R^n,(x_i )_{i=1}^\infty 是 A 中的点列， x=(x_1,x_2,\cdots,x_n)\in A 。记点列中的每一项 x_i=(x_{1,i},x_{2,i},\cdots,x_{n,i}) ，则对于每个分坐标 x_{k,i},k=1,2,\cdots,n ，我们得到了 n 个 \mathbb R 中的点列 (x_{k,i})_{i=1}^\infty 。此时以下两个命题等价：

  1. (x_i )_{i=1}^\infty\rightarrow x 。
  2. 对于每个 k=1,2,\cdots,n ，点列 (x_{k,i})_{i=1}^\infty\rightarrow x_k 。

证明：先证 1\Rightarrow2 。设 (x_i )_{i=1}^\infty\rightarrow x ，则 \forall\varepsilon>0,\exists N\in\mathbb N_+ ，当 i\geq N 时， d(x_i,x)=||x_i-x||<\varepsilon ，这里的 ||\: || 表示欧几里得范数。任取某个坐标分量 x_{k,i} ，则 |x_{k,i}-x_k|=\sqrt{(x_{k,i}-x_k)^2}\leq\sqrt{(x_{1,i}-x_1)^2+(x_{2,i}-x_2)^2+\cdots+(x_{n,i}-x_n)^2}=||x_i-x|| 。所以当 i\geq N 时， |x_{k,i}-x_k|\leq ||x_i-x||<\varepsilon 成立，因此 (x_{k,i})_{i=1}^\infty\rightarrow x_k 。

再证 2\Rightarrow1 。 \forall\varepsilon>0 ，因为对于每个 k=1,2,\cdots,n ，点列 (x_{k,i})_{i=1}^\infty\rightarrow x_k ，所以 \exists N_k\in\mathbb N_+ ，当 i \geq N_k 时， |x_{k,i}-x_k|<\frac{\varepsilon}{\sqrt{n}} 。取 N=\max\left\{ N_1,N_2,\cdots,N_n \right\} ，则当 i\geq N 时， ||x_i-x||^2=(x_{1,i}-x_1)^2+(x_{2,i}-x_2)^2+\cdots+(x_{n,i}-x_n)^2<\frac{\varepsilon^2}{n}\cdot n=\varepsilon^2 ，因此 d(x_i,x)=||x_i-x||<\varepsilon ，这说明 (x_i )_{i=1}^\infty\rightarrow x 。

证毕。
